# LLM Wiki 目次

この wiki は、一次資料をもとに LLM が継続的に保守する知識ベースです。入口は概念マップに限定し、個別の concept page や source page へは各 map page から降ります。

## 概念マップ

- [[maps/llm-wiki-operations|LLM Wiki 運用概念マップ]] - この wiki を構成する source page、concept page、map page、ingest、query、lint の関係。
- [[maps/ab-testing|A/B テスト概念マップ]] - 実験、意思決定、サンプルサイズ、時間依存指標の関係。
- [[maps/causal-inference|因果推論概念マップ]] - 潜在的結果フレームワーク、処置効果推定、観察データ、機械学習手法の関係。
- [[maps/statistical-modeling|統計モデリング概念マップ]] - 因果推論、予測不確実性、ロバスト性、高次元統計解析、階層ベイズモデル、逐次意思決定の関係。
- [[maps/category-theory|圏論概念マップ]] - 圏論の基本語彙、確率論への応用、計算効果を表すモナドの関係。
- [[maps/type-theory-and-type-inference|型理論と型推論概念マップ]] - 型理論、パラメトリック多相、局所的型推論、大域的型推論、非可述的多相性の関係。
- [[maps/recommender-systems|推薦システム概念マップ]] - 推薦システム、潜在因子モデル、行列分解、学習データ、最適化の関係。
- [[maps/graph-theory|グラフ理論概念マップ]] - 整数グラフ、ケイリーグラフ、circulant グラフ、エクスパンダー、lamplighter graph の関係。
- [[maps/social-cognition|社会的認知概念マップ]] - 名前による印象形成、奇抜な名前、社会的評価、リーダーシップ性評価の関係。
